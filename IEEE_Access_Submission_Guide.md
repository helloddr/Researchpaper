# IEEE Access Submission Guide

## Recommended Target
Recommended first target for this paper: IEEE Access.

Reason:
- broad scope covering AI, machine learning, and healthcare applications
- rolling submission model
- more suitable for this work than a narrow top-tier specialty venue

## Files To Use
Use these files as the current submission package:
- Final_Research_Paper_Review.docx
- Final_Research_Paper_Review.pdf
- IEEE_Submission_Checklist.txt
- README.md

Do not use the older `Final_IEEE_Paper_Draft*` files.

## Before Upload
1. Run plagiarism or similarity check on `Final_Research_Paper_Review.pdf`.
2. Review the DOCX once in Word for page breaks, tables, and figure readability.
3. Download the official IEEE Access Word template and paste the final content into it if the portal requires strict formatting.
4. Keep the datasets, code, and outputs available in case the journal asks for reproducibility details.

## Important Submission Notes
- Submission is done through the journal portal, not through a general IEEE upload page.
- The author must complete all declarations, copyright, and funding/conflict statements personally.
- If AI tools were used in manuscript drafting or editing, disclosure should remain in the manuscript and match the journal policy.
- Final acceptance depends on editor and reviewer evaluation, not only on formatting.

## Practical Upload Order
1. Prepare the final PDF.
2. Prepare the final DOCX source.
3. Prepare title, abstract, keywords, and author information.
4. Log in to the submission portal.
5. Fill all metadata fields carefully.
6. Upload manuscript files.
7. Review the generated proof before final submission.
